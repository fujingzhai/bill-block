# 记账块

记账块是在思源文档中直接嵌入的轻量记账挂件，用于在日记、月度复盘、项目预算或个人工作台里随手记录和查看支出。

它不是独立的记账 App，也不是数据库视图的替代品。它关注一个更窄、更顺手的场景：不用建数据库、不用离开当前文档，就能在笔记里放一个可编辑、可长期保存、多端同步的记账块。全部记账块共享同一份账单数据，在任意一个块里增、改、删，其他块即时更新。

## 统一记账块

通过斜杠菜单 `/记账块`（或输入「记账」「账单」「jizhang」）、顶栏 ¥ 菜单中的「插入记账块」插入，可在任意文档插入任意多个。每个块内置四个视图，块内切换，互不干扰：

- **流水**：顶部快记行（金额 → 分类 → 项目 → 备注 → 日期 → 回车），下方按日期分组、倒序排列的全部流水，支持行内编辑与删除。流水是唯一允许出现滚动条的视图，高度与周账对齐，列表在块内滚动而不会把整块撑长。
- **周账**：本周一至周日的七日板，按当日支出深浅着色；点击某天查看明细，点击格内「＋」或双击条目即可就地记账、改账。下方附「分类」柱状图与「走势」折线图。
- **月账**：整月日历板，逻辑与周账一致；每个格子最多直显三条，超出折叠为「＋N」。
- **统计**：分类水平条形图（含占比，可按默认 / 降序 / 升序排序）与走势折线图（可按日 / 按周 / 按月 / 按年，自定义时间范围，多分类同时显示）。

## 核心功能

- 一份账单数据，多个块、多个视图实时同步，不依赖思源数据库。
- 新账本默认不预置任何分类，由用户按自己的记账习惯从零建立。
- 账单存放在独立的工作空间文件中，随思源官方云同步多端同步。
- 流水 / 周账 / 月账顶部均有快记行，金额与项目必填，分类、备注、日期可选。
- 周账 / 月账可把某一周 / 某一月设为本块默认时间并写入块属性，重载后保留；左右翻看后点中间标题一键跳回。
- 默认周选择器支持「年 + 周次」直接输入或日历选择，并就近括注该周的具体日期范围。
- 周账 / 月账格内条目默认显示项目名；写了备注时，鼠标悬停条目即以浮层提示备注，没有备注则不提示。
- 统计走势折线图：总支出用中性灰线，各分类用自己的颜色；多条折线时，鼠标悬停某条折线的点会高亮该线、淡出其余，并配合放大的命中区域，方便区分重叠折线。
- 分类柱状图默认显示全部分类；可在分类管理中勾选「隐藏无支出的分类」，只看本期有支出的分类。
- 两级分类管理：新增、重命名、换色、拖拽排序、删除（删除后相关账目归入「未分类」）；提供 10 色板、调色板取色与 `#RRGGBB` 色码自定义。
- 全局快捷键 `Option+Command+B`（可在思源设置中修改）随处弹出「记一笔」；顶栏 ¥ 菜单也可快速记一笔。
- 每个视图右上角可一键将当前视图截图导出为 PNG，导出时自动隐藏设置类按钮等悬浮控件，并优先以桌面作为保存位置。
- 支持 `Cmd/Ctrl+Z` 撤销最近一次账单操作（最多 50 步）。

## 交互细节

- 快记行与行内表单在金额、项目合法时回车即保存；按 `Esc` 或点「取消」放弃本次输入。
- 默认周、默认月按钮，以及分类管理按钮，均支持再次点击同一按钮关闭弹层。
- 行内编辑时点击块外（包括跨 iframe 点到思源主界面）或块内空白处会自动退出编辑、恢复明细。
- 周账 / 月账格子按当日支出额深浅着色，今天与选中日有描边强调。
- 金额按本地习惯千分位显示，统一使用等宽数字对齐。

## 数据存储

账单数据统一保存到思源工作空间：

`/data/storage/bill-block/data.json`

每次写入前会把上一份正确数据备份到同目录的 `data.json.bak`；当文件损坏或读取异常时自动禁止写入，避免覆盖原有数据。多个块同时打开时通过 BroadcastChannel 即时同步，跨设备依赖思源官方云同步。

记账块自身还会在块属性中保存本块的默认周 / 默认月，用于下次打开时恢复。

## 适用场景

- 日记、月度与年度复盘里的随手记账。
- 项目文档中的预算与花销记录。
- 旅行、装修、活动等一次性事项的开销清单。
- 个人工作台中的支出概览。
- 任何希望把账目直接放在笔记内容里、又不想另开记账软件的场景。
